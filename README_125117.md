# PROJETO PYTHON #

# Objectivo #
Desenvolver um sistema que permite implementar e demonstrar o funcionamento das estruturas de dados pilhas e filas utilizando a linguagem python.


# Funcionalidades #
- Inserção e remoção de elementos.
- Consulta de elementos.
- Visualizar estruturas vazias.
- Contagem de elementos.
- Simulação prática de elementos.

# Tecnologias #
- Python
- Programação Orientada a Objectos

  # Disciplina #
  Algoritmo e Estrutura de Dados
